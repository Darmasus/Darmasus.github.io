ENHANCEMENT ONE

Software Design and Engineering

The software design enhancement focused on improving maintainability, readability, validation, secure design, and separation of concerns. I added reusable helper classes and updated the activity logic so validation and security responsibilities were no longer mixed directly into screen code.

  • Created ValidationHelper.java for reusable validation checks.
  • Created SecurityHelper.java for password hashing responsibilities.
  • Updated LoginActivity.java to improve login and account validation.
  • Updated DashboardActivity.java to improve weight validation and error handling.
  • Improved defensive programming by checking database operation results.

ENHANCEMENT TWO

Algorithms and Data Structures

The algorithms and data structures enhancement improved how the application processes stored weight data. Instead of only displaying individual records, the application now performs grouped analysis, trend detection, and structured collection processing.

  • Added weekly average calculations in DatabaseHelper.java.
  • Used SQL aggregation to group and calculate weight averages.
  • Used LinkedHashMap and Map traversal to process grouped data.
  • Added trend detection logic in DashboardActivity.java.
  • Improved sorting by using the weight date column instead of insertion order.

ENHANCEMENT THREE

Databases

The database enhancement focused on improving data integrity, query performance, and reliability. I added database indexes and duplicate-entry prevention so the application can better protect the accuracy of stored weight records.

  • Added indexes for username, date, and combined username/date lookups.
  • Added weightEntryExists() in DatabaseHelper.java.
  • Updated DashboardActivity.java to prevent duplicate entries for the same user and date.
  • Improved database integrity before inserting new records.
  • Improved the database layer for future scalability.
